# Dakshin Chakra WordPress Theme

Upload `wp-content/themes/dakshin-chakra` to your WordPress installation and activate it.

## Included
- Full block theme templates, header/footer, SEO and schema helpers.
- Automatic non-destructive page seeding for required pages.
- Static front page assignment to **Home** on activation.
- Floating WhatsApp CTA using `https://wa.me/916364659339`.

## Gutenberg Editing
All pages render via `post-content` blocks in templates so content remains editable in Gutenberg.

## Control points
- WhatsApp number/link: `parts/footer.html` and seed CTA links in `inc/page-seeds.php`.
- Header: `parts/header.html`
- Footer: `parts/footer.html`

## Manual WordPress Admin Steps
1. Activate theme.
2. Verify permalinks and page content in **Pages**.
3. Optionally replace placeholder image panels in Gutenberg.
